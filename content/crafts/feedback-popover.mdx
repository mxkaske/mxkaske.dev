The Feedback Popover is inspired by [Resend](https://resend.com)'s feedback form. Designed with [ui.shadcn](https://ui.shadcn.com).

Install the `@types/dom-speech-recognition` types package to get the `SpeechRecognition` type.

```bash
npm install --save-dev @types/dom-speech-recognition
```

Exchange `const [status, setStatus] = useState<Status>("idle");` with your own actions to send the feedback to your slack channel e.g.

Depending on the location of the popover, you might need to adjust the `align` and/or `side` prop of the `PopoverContent` component.
